# Electrician Helper

Console-based helper program written in C

## Features Planned
    - Ohm's Law calculator
    - Power calculations
    - Voltage drop calculations
    - Amp draw calculations
    - Series/parallel resistance
    - Calculation history
    - CSV/text export

## Build

Compile with GCC:

```bash
gcc ElectricianHelper.c -o ElectricianHelper
```

Run:

```bash
./ElectricianHelper
```
